# Small

The _Small_ tag defines smaller text (like copyright and other side-comments).

## Usage

```javascript
import { Small } from 'quarks-ui';
```

## Best practices

-   This tag is not deprecated, but it is possible to achieve richer (or the same) effect with CSS.

<br/>

## Customization

### Color

The color utility parses a component's _color_ and _bg_ props and converts them into CSS declarations. By default the raw value of the prop is returned. Color palettes can be configured with the ThemeProvider to use keys as prop values, with support for dot notation.

You can see all the color customization [here](https://styled-system.com/api/#color).

```javascript

// picks the value defined in `theme.colors.blue`
<Small styling="base" color='blue'>small</Small>

// picks up a nested color value using dot notation `theme.colors.gray[0]`
<Small color='gray.0'>small</Small>

// raw CSS color value
<Small color='#f00'>small</Small>

// background colors
<Small bg='blue'>small</Small>

// verbose prop
<Small backgroundColor='blue'>small</Small>
```

### Space

The space utility converts shorthand margin and padding props to _margin_ and _padding_ CSS declarations.

You can see all the space customization [here](https://styled-system.com/api#space).

```javascript

// sets margin value of `theme.space.md`
<Small styling="base" m="md">small</Small>

// sets a margin value of `16px` since it's greater than `theme.space.length`
<Small m={16}>small</Small>

// sets margin `'auto'`
<Small m='auto'>small</Small>

// sets margin `8px` fom mobile devices, `16px` for tablet and `32px` for desktop from the first breakpoint and up
<Small m={{ mobile:2, tablet:3, desktop:4 }}>small</Small>
```

### Typography

Typography props include _fontFamily_, _fontSize_, _fontWeight_, _lineHeight_, _letterSpacing_, _textAlign_, and _fontStyle_.

You can see all the typography customization [here](https://styled-system.com/api/#typography).

```javascript

// font-size of `theme.fontSizes[8]`
<Small styling="base" fontSize={8}>small</Small>

// font-size `32px`
<Small fontSize={32}>small</Small>

// font-size `'2em'`
<Small fontSize='2em'>small</Small>

// font-size `10px` on all viewports and `12px` from the first breakpoint and up
<Small fontSize={{ mobile:10, desktop:12 }}>small</Small>

// fontFamily
<Small fontFamily='mono'>small</Small>

// textAlign
<Small textAlign='center'>small</Small>
<Small textAlign={{ mobile: 'center', tablet: 'left', desktop: 'center' }}>small</Small>

// lineHeight
<Small lineHeight='1.25'>small</Small>

// fontWeight
<Small fontWeight='bold'>small</Small>

// letterSpacing
<Small letterSpacing='0.1em'>small</Small>
```
