# P

The HTML _P_ element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

### Best practices

-   You cannot be sure how HTML will be displayed.
-   Large or small screens, and resized windows will create different results.
-   With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code.
-   The browser will automatically remove any extra spaces and lines when the page is displayed

## Customization

### Color

The color utility parses a component's _color_ and _bg_ props and converts them into CSS declarations. By default the raw value of the prop is returned. Color palettes can be configured with the ThemeProvider to use keys as prop values, with support for dot notation.

You can see all the color customization [here](https://styled-system.com/api/#color).

```javascript
// picks the value defined in `theme.colors.blue`
<P styling="base" color='blue'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// picks up a nested color value using dot notation `theme.colors.gray[0]`
<P color='gray.0'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// raw CSS color value
<P color='#f00'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// background colors
<P bg='blue'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// verbose prop
<P backgroundColor='blue'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>
```

### Space

The space utility converts shorthand margin and padding props to _margin_ and _padding_ CSS declarations.

You can see all the space customization [here](https://styled-system.com/api#space).

```javascript
// sets margin value of `theme.space[2]`
<P styling="base" m={2}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// sets a margin value of `16px` since it's greater than `theme.space.length`
<P m={16}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// sets margin `'auto'`
<P m='auto'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// sets margin `8px` fom mobile devices, `16px` for tablet and `32px` for desktop from the first breakpoint and up
<P m={{ mobile:2, tablet:3, desktop:4 }}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>
```

### Typography

Typography props include _fontFamily_, _fontSize_, _fontWeight_, _lineHeight_, _letterSpacing_, _textAlign_, and _fontStyle_.

You can see all the typography customization [here](https://styled-system.com/api/#typography).

```javascript
// font-size of `theme.fontSizes[3]`
<P styling="base" fontSize={3}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// font-size `32px`
<P fontSize={32}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// font-size `'2em'`
<P fontSize='2em'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// font-size `10px` on all viewports and `12px` from the first breakpoint and up
<P fontSize={{ mobile:10, desktop:12 }}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// fontFamily
<P fontFamily='mono'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// textAlign
<P textAlign='center'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>
<P textAlign={{ mobile: 'center', tablet: 'left', desktop: 'center' }}>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// lineHeight
<P lineHeight='1.25'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// fontWeight
<P fontWeight='bold'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>

// letterSpacing
<P letterSpacing='0.1em'>Lorem ipsum, dolor sit amet consectetur adipisicing elit.</P>
```
