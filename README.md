## Quick start

Several quick start options are available:

-   Clone the repo:

```
  git clone https://github.com/juniorconquista/quarks-ui.git
```

-   Install with [npm](https://www.npmjs.com/):

```
  npm install quarks-ui
```

-   Install with [yarn](https://yarnpkg.com/):

```
  yarn add quarks-ui
```
