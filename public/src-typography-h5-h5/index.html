Getting Started

H5

Search engines use the headings to index the structure and content of your web pages. Users often skim a page by its headings. It is important to use headings to show the document structure.

Each HTML heading has a default size. However, you can specify the size for any heading with the style attribute, using the CSS font-size property.

Best practices

  • Use HTML headings for headings only. Don't use headings to make text BIG or bold.

Properties

Basic usage

Title
<H5 styling="base">Title</H5>

Customization

Color

The color utility parses a component's color and bg props and converts them into CSS declarations. By default the raw value of the prop is returned. Color palettes can be configured with the ThemeProvider to use keys as prop values, with support for dot notation.

You can see all the color customization here.

Title
<H5 styling="base" color="blue">
  Title
</H5>
// picks the value defined in `theme.colors.blue`
<H5 styling="base" color='blue'>Title</H5>
// picks up a nested color value using dot notation `theme.colors.gray[0]`
<H5 color='gray.0'>Title</H5>
// raw CSS color value
<H5 color='#f00'>Title</H5>
// background colors
<H5 bg='blue'>Title</H5>
// verbose prop
<H5 backgroundColor='blue'>Title</H5>

Space

The space utility converts shorthand margin and padding props to margin and padding CSS declarations.

You can see all the space customization here.

Title
<H5 styling="base" m={2}>
  Title
</H5>
// sets margin value of `theme.space[2]`
<H5 styling="base" m={2}>Title</H5>
// sets a margin value of `16px` since it's greater than `theme.space.length`
<H5 m={16}>Title</H5>
// sets margin `'auto'`
<H5 m='auto'>Title</H5>
// sets margin `8px` fom mobile devices, `16px` for tablet and `32px` for desktop from the first breakpoint and up
<H5 m={{ mobile:2, tablet:3, desktop:4 }}>Title</H5>

Typography

Typography props include fontFamily, fontSize, fontWeight, lineHeight, letterSpacing, textAlign, and fontStyle.

You can see all the typography customization here.

Title
<H5 styling="base" fontSize={2}>
  Title
</H5>
// font-size of `theme.fontSizes[2]`
<H5 styling="base" fontSize={2}>Title</H5>
// font-size `32px`
<H5 fontSize={32}>Title</H5>
// font-size `'2em'`
<H5 fontSize='2em'>Title</H5>
// font-size `10px` on all viewports and `12px` from the first breakpoint and up
<H5 fontSize={{ mobile:10, desktop:12 }}>Title</H5>
// fontFamily
<H5 fontFamily='mono'>Title</H5>
// textAlign
<H5 textAlign='center'>Title</H5>
<H5 textAlign={{ mobile: 'center', tablet: 'left', desktop: 'center' }}>Title</H5>
// lineHeight
<H5 lineHeight='1.25'>Title</H5>
// fontWeight
<H5 fontWeight='bold'>Title</H5>
// letterSpacing
<H5 letterSpacing='0.1em'>Title</H5>
